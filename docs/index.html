Philosophy

DeepL CLI embraces the core tenets of the Unix philosophy:

  • Do one thing and do it well: Its primary function is text translation via DeepL API.
  • Work together: Designed to be easily integrated with other command-line tools through pipes (e.g., `cat file.txt | deepl | less`).
  • Text is the universal interface: Input and output are plain text, making it highly interoperable.
  • Build a prototype as soon as you can: Focus on core functionality first, then iterate.

About DeepL CLI

DeepL CLI is a powerful command-line tool that allows you to leverage the DeepL API for high-quality text translations directly from your terminal. It's built with Go, ensuring fast performance and easy cross-platform compatibility.

For more details, visit the DeepL CLI GitHub repository.
